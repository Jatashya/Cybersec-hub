About The Project: Aegis Hub

Aegis Cybersecurity Hub is a modern, responsive Single Page Application (SPA) built to empower users with the tools needed to combat daily digital threats. By integrating real-time threat analysis with interactive education, Aegis bridges the gap between passive learning and active defence.

Core Modules Developed:

  • Security Dashboard: Aggregates live user data to calculate a Personal Cyber Safety Index and visualizes risk through a local network threat shield.
  • Password Analyzer: A cryptographic analysis tool that tests Shannon entropy, highlights structural weaknesses, and calculates crack durations in real-time.
  • URL Phishing Check: An inspector running heuristic algorithms to detect typosquatting, suspicious TLDs, and deceptive redirects.
  • Email Scam Scanner: A threat-level scraper utilizing regex and keyword detection to highlight psychological manipulation vectors (like financial urgency).
  • Interactive Learning Hub & Quiz: Features interactive flashcards, a unique "Spot the Phish" simulation engine, and an exam system to test readiness and reward safe practices.
